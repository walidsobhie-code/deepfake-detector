# 🛠️ Setup Guide

## Prerequisites
- Python 3.10+
- Git

## Environment Setup
1. Clone the repository
2. Run `pip install -r requirements.txt`
3. Copy `.env.example` to `.env`
4. Add your API keys

## Common Issues
**Module not found** → Run `pip install -r requirements.txt`
**API key error** → Check your `.env` file
